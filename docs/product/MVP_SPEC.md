# Fencepac Knowledge Platform MVP Spec

## Version

v0.1 - Local Knowledge Browser

## Main pages

### Home
Dashboard with:
- product categories
- number of product topics
- number of sources
- training module card
- warnings for draft content

### Wall Systems
Grid of product/system cards.

Each card shows:
- name
- category
- system type
- short description
- source status

### Product detail
Each product page shows:
- overview
- typical use
- system type
- key components
- installation basics
- QA checks
- common risks
- source cards

### Sources
Source register showing:
- title
- supplier/source owner
- source type
- status
- URL or local file placeholder
- linked product topic

### Glossary
Retaining wall terminology:
- gravity wall
- MSE wall
- RSS
- geogrid
- surcharge
- bearing capacity
- friction angle
- cohesion
- drainage layer
- ag pipe
- geotextile
- hold point

### Training
One module:
Retaining Wall Basics

Lessons:
1. Wall families
2. Gravity vs reinforced earth
3. Drainage basics
4. Soil and surcharge basics
5. Product overview
6. Site risks
